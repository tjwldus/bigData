# 🚗 교통사고 데이터 분석 프로젝트

## 1. 프로젝트 설명
이 프로젝트는 공공데이터포털에서 제공하는 **한국도로교통공단_시도 시군구별 교통사고 통계 데이터**를 기반으로,
Streamlit 웹 대시보드를 제작하여 지역별 교통사고 현황을 시각화하고 분석하는 것을 목표로 한다.

사용자는 시도 및 시군구별 교통사고 규모, 사망자 수, 부상자 수 등을 확인할 수 있으며,
사망률/경상률 분석 및 시도 간 비교 분석을 통해 지역별 사고 특성을 파악할 수 있다.

---

## 2. 주제
**한국 교통사고 데이터 시각화 및 지역별 사고 특성 분석**

---

## 3. 주제 선택 이유
- 국내 교통사고는 매년 심각한 인명 피해를 유발하는 중요한 사회 문제이다.
- 지역(시도·시군구) 특성에 따라 사고 발생 패턴과 위험도가 다르다.
- 데이터를 기반으로 시각∙통계 분석을 수행하면 사고 예방 정책 및 안전 개선 방향을 도출할 수 있다.
- Streamlit을 활용하면 누구나 쉽게 접근해 볼 수 있는 대시보드 형태로 제공할 수 있다.

---

## 4. 데이터 분석 내용

### 📁 사용 데이터
- 데이터 출처: 공공데이터포털  
- 데이터명: *한국도로교통공단_시도 시군구별 교통사고 통계*  
- 주요 컬럼:  
  - 시도  
  - 시군구  
  - 사고건수  
  - 사망자수  
  - 중상자수  
  - 경상자수  
  - 부상신고자수  

### 📌 분석 기능
- 시도별 사고 통계 시각화
- 시군구별 사고 지도/그래프 분석
- 시도 간 교통사고 지표 비교
- 사망률 vs 경상률 산점도 분석  
  - 사망률 = 사망자수 / 사고건수  
  - 경상률 = 경상자수 / 사고건수
